# 🤖 AI Search Agent

A conversational AI assistant built using:

- LangChain
- Google Gemini
- DuckDuckGo Search
- Streamlit

## Features

- Conversational AI
- Web Search using DuckDuckGo
- Chat History
- Modern Streamlit UI

## Installation

```bash
pip install -r requirements.txt
```

Run

```bash
streamlit run app.py
```

## Technologies

- LangChain
- Gemini 2.5 Flash
- DuckDuckGo Search
- Streamlit
